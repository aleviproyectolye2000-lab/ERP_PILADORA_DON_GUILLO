Esta opción se usa solamente cuando la empresa decide subir o corregir el sueldo base de un empleado. El cambio debe quedar validado y auditado.

Asistencia manual de empleados

Registro manual de entrada y salida mientras no exista sensor biométrico. Las horas extras y sanciones registradas aquí se usarán automáticamente para generar el rol de pago.

Sanción: puede ser 0% cuando no aplica, o de 1% a 100% cuando la empresa decida aplicar descuento por atraso, falta u otra observación.

Generar rol de pago

El rol de pago se genera automáticamente usando el sueldo del empleado, las horas extras, sanciones registradas en asistencia y el aporte obligatorio IESS del 9.45%. Las bonificaciones y descuentos adicionales se ingresan manualmente si aplican.

Regla de cálculo: cada hora extra vale por defecto $10.00. Si se desea modificar ese valor, debe validarse con clave de autorización. El aporte IESS del empleado se calcula automáticamente con el 9.45%.
