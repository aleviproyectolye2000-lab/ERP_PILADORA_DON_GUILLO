# Backend - ERP Piladora Don Guillo

## 1. Descripción

Este backend pertenece al sistema ERP de la Piladora Don Guillo. Está desarrollado con Python y FastAPI, y permitirá gestionar los módulos de compras, recepción de arroz en cáscara, inventario, producción, ventas, reportes, usuarios, auditoría e inteligencia artificial para toma de decisiones.

## 2. Herramientas utilizadas

- Python 3.12.5
- FastAPI
- Uvicorn
- PostgreSQL 17.10
- SQLAlchemy
- psycopg2-binary
- python-dotenv
- Pydantic
- ReportLab
- openpyxl
- pandas
